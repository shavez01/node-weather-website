# Node Weather Website

A simple weather web application built with **Node.js**, **Express**, and **Handlebars** that lets users look up real-time weather information by city or address.

This project includes server-side geocoding and weather API integration so you can get live weather data from the browser.

---

## 🧠 Features

✔ Enter a city or address to get current weather information  
✔ Built with Node.js and Express for backend APIs  
✔ Uses third-party API services for geolocation and weather data  
✔ Dynamic templates with Handlebars  
✔ Responsive, clean UI  

*Inspired by common weather app tutorials and full-stack Node projects.* :contentReference[oaicite:1]{index=1}

---

## 📦 Technologies Used

✔ **Node.js** – JavaScript runtime  
✔ **Express** – Web framework  
✔ **Handlebars (hbs)** – Templating engine  
✔ **Mapbox API** (or similar) – Geocode lookup  
✔ **Weather API** (e.g., Weatherstack / OpenWeatherMap) – Weather data

## 🚀 How to Run Locally

### 1. Clone the repository  
```bash
git clone https://github.com/shavez01/node-weather-website.git
```

2. Install dependencies
```bash
npm install
```

3. Add API keys
```code
MAPBOX_API_KEY=your_mapbox_key
WEATHER_API_KEY=your_weather_api_key
```

4. Start the server
```bash
npm start
```
